# Level 3 Task 2 — Async Programming & Multithreading

6 comprehensive demos covering all async/threading concepts.

## Demos
1. **async/await** — concurrent API calls, WhenAll vs sequential
2. **Task Parallel Library** — Parallel.ForEach, PLINQ
3. **ThreadPool & CancellationToken** — background work + cancellation
4. **Race Conditions** — unsafe counter vs lock vs Interlocked
5. **SemaphoreSlim** — limit concurrent operations (connection pool)
6. **Producer-Consumer** — BlockingCollection pattern
