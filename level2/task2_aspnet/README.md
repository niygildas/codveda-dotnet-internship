# Level 2 Task 2 — ASP.NET Core Blog Platform

Full MVC blog with REST API, Entity Framework Core (SQLite), Repository Pattern.

## Features
- CRUD for blog posts and comments
- MVC views with Razor syntax
- REST API at /api/BlogApi
- SQLite database with EF Core
- Swagger UI at /swagger
- Dependency injection

## Run
```bash
dotnet run
```
Open: http://localhost:5000
